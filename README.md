# Tabbed.io
The personalised landing page for your browser

![The Tabbed.io New Tab Page](http://i.imgur.com/qUVw3yo.jpg)

## Features

* Randomise the wallpaper
* Time and date settings
* Countdown timer
* Google search
* Custom wallpaper from a URL
